# Dossier fonts

Used only by `scripts/build-dossier.mjs` to render the brand PDF — not bundled
into the web app (the site loads Fraunces/Geist via their npm packages).

- **Fraunces** (`Fraunces-*.ttf`) — display serif, © The Fraunces Project Authors,
  licensed under the SIL Open Font License 1.1 (https://openfontlicense.org).
  Static instances extracted from `@expo-google-fonts/fraunces`.
- **Geist** — body/UI sans, loaded from the `geist` npm package (also OFL).
